# notebooks/

Exploratory analysis and pipeline development. Notebooks here are for exploration and prototyping — once logic is settled, it gets promoted into a repeatable script under `../scripts/`.

## Naming convention

`NN_short-description.ipynb`, numbered in the order they're meant to be read/run, e.g.:

- `01_chicago_data_exploration.ipynb`
- `02_nyc_data_exploration.ipynb`
- `03_sf_data_exploration.ipynb`
- `04_cross_city_schema_mapping.ipynb`

Populated starting in Milestone 1.
